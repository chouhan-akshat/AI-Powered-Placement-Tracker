# 🚀 AI Powered Placement Tracker

An intelligent platform that helps students prepare for placements with a structured roadmap, progress tracking, and AI-driven guidance.

---

## 🧠 Problem
Many students struggle with:
- ❌ No clear roadmap  
- ❌ Random preparation  
- ❌ No progress tracking  
- ❌ Lack of interview practice  

---

## 💡 Solution
This platform acts as a **personal placement mentor** that:
- Generates **AI-based personalized roadmaps**
- Tracks daily/weekly progress  
- Provides **AI mock interviews**  
- Gives performance insights  

---

## ⚙️ Features
- 📌 Personalized study roadmap (DSA, Aptitude, Core)
- 📊 Progress tracking dashboard  
- 🤖 AI-powered mock interviews  
- 🧩 Task & goal management  
- 📈 Performance analytics  

---

## 🛠️ Tech Stack
- **Frontend:** React / HTML / CSS / JavaScript  
- **Backend:** Node.js / Express  
- **Database:** MongoDB  
- **AI:** OpenAI API / Gemini API
